# NyayaBot ⚖️

NyayaBot is an AI-powered legal assistance chatbot that helps victims understand applicable Indian laws and legal actions.

## Features
- Crime detection from user incident description
- Relevant BNS / IT Act sections
- FIR draft generator
- Crime severity indicator
- Multilingual support (English, Hindi, Hinglish)
- Crime awareness mode

## Tech Stack
- AI Chatbot
- Natural Language Processing
- Antigravity AI

## How It Works
User describes an incident → NyayaBot analyzes it → suggests legal sections and next steps.
